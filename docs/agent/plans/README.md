# Execution Plans

Use this directory for complex work whose state must survive a chat session.

- Start from `_template.md`.
- Put ongoing plans in `active/`.
- Update progress and decisions while working.
- Move finished plans to `completed/` without erasing deviations or unresolved follow-up work.
- Do not create a checked-in plan for a trivial, reversible local change.

File names use `YYYY-MM-DD-short-topic.md`.
